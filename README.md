# ミテルンデス Android版 🔍

**保存できない拡大カメラアプリ** — 美術館・博物館・劇場で安心して使える、プライバシー重視の拡大鏡

[![Kotlin](https://img.shields.io/badge/Kotlin-1.9-purple)](https://kotlinlang.org)
[![Android](https://img.shields.io/badge/Android-API%2026+-green)](https://developer.android.com)

## 概要

ミテルンデスは、視覚障害者や高齢者が**文化施設で安心して使える拡大カメラアプリ**です。iOS版の完全再現を目指したAndroid版で、Kotlin + Jetpack Composeで開発しています。

## ✨ 主な機能

- 📷 **高倍率ズーム** — ピンチ＆ボタンで最大200倍まで拡大
- ⏱ **10分自動削除** — 撮影画像は10分後に自動で消去
- 🛡 **スクリーンショット・画面録画防止** — 不正コピーをブロック
- 🎭 **シアターモード** — 美術館・劇場向けの静音・暗画面モード
- 📰 **プレスモード** — 報道関係者向け認証付きモード
- 🌊 **透かし（ウォーターマーク）** — 保存画像にウォーターマーク付与
- 🌐 **日本語 / 英語対応**
- ♿ **TalkBack完全対応** — アクセシビリティファースト設計

## 🛠 技術スタック

- **言語:** Kotlin
- **UI:** Jetpack Compose
- **カメラ:** CameraX
- **アーキテクチャ:** MVVM
- **対応OS:** Android 8.0+ (API 26)
